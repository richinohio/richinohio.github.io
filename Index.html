Summary: Latitude vs. X

This project sought to use weather data to show how temperatures varied as you got closer to the equator.

To do this, we used the OpenWeatherMap API to assemble a data set that contained data for more than 500 cities around the world.

We then used Matplotlib library to plot various aspects of weather vs. Latitude. Thesee factors included max temperature, humidity, cloudiness and wind speed.

This site summarizes our work, and includes the source data, the visualizations and some of the key conclusions that we drew from the analysis.
